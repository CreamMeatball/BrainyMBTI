TBTI - 더 브레이니 성향 검사

친구의 TBTI는?

I (Introversion) : 내향적이고 신중함
E (Extrovert) : 외향적이고 주도적

N (North) : 강남보다 강북을 선호
S (South) : 강북보다 강남을 선호

T (True) : 솔직하고 정정당당한 타입
F (False) : 정치와 속임수를 즐기는 타입

J (Jeul-game) : 가벼운 재미와 흥미를 추구
P (Ppak-game) : 치열한 경쟁과 두뇌싸움을 추구

A (Alcohol) : 음주와 술자리를 모두 즐김
N (Non-alcohol) : 음주는 즐기지 않고 술자리만 즐김

[TBTI - The Brainy Type Indicator] 더 브레이니 성향 검사

제작 : 1기 김경린 / 5기 박철현 / 1기 이준호
